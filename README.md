# Stanley Otano Portfolio

Recreated static portfolio website for Stanley Otano.

## Files

- `index.html` — website content
- `style.css` — styling and responsive layout
- `script.js` — mobile menu behavior
- `_redirects` — Netlify fallback rule

## Deploy on Netlify

1. Log in to Netlify.
2. Choose **Add new site**.
3. Connect this GitHub repository or deploy manually.
4. For a static HTML site, no build command is needed.
5. Publish directory should be the repository root.

## Notes

The current image links load from the old live Netlify portfolio. For a fully independent rebuild, replace those external image URLs with local images inside this repository.

The resume button points to `resume.pdf`. Add a `resume.pdf` file in the root if you want that button to work.
